# Лабораторная работа: метод Стронгина для глобального поиска минимума

## Цель

Реализовать на языке Python алгоритм глобальной минимизации одномерной
липшицевой функции по методу Стронгина и проверить его работу на функции
с несколькими локальными минимумами.

В качестве примера используется одномерный вариант функции Растригина.

---

## Что делает программа

Скрипт `main.py` решает задачу:

> найти приближённое значение глобального минимума функции `f(x)` на отрезке `[a, b]`.

Функция задаётся строкой `FUNCTION_STRING`, а границы отрезка и точность
определяются константами в начале файла.

На выходе программа:

- вычисляет приближённое положение минимума `x*` и значение `f(x*)`;
- показывает в консоли:
  - число итераций алгоритма;
  - сколько раз была вычислена функция;
  - затраченное время;
- строит график:
  - исходной функции;
  - точек испытаний (выборки метода);
  - вспомогательной ломаной — нижней оценки, используемой в методе Стронгина.

## Установка и запуск

Зависимости:

```bash
pip install numpy matplotlib
```

Или через файл:
```bash
pip install -r requirements.txt
```

Запуск программы:

```bash
python main.py
```
